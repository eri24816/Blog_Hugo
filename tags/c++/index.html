    Reverb plugin

    March 16, 2022

    一直想做的東西後來真的有課教我怎麼做，就會覺得那個課很棒。 在「數位音樂訊號分析」這門課中，十幾個學生分成6組，每組使用 JUCE(一個製作 VST 的 C++ 框架)來做一種 VST 效果器。我們這組做的是 reverb。Reverb 的功用是把音樂加上迴音，像是在大教堂或音樂廳的感覺。
